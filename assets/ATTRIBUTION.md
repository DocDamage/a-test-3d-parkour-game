# Asset Attribution

Runtime assets in this folder are converted from local source packs under `my game assets/`.

Known source notes:

- `assets/models/enemies/eye_monster.glb` is based on "Animated FPS Eye Monster" by Levraicoincoin on Sketchfab, licensed under CC-BY-4.0.
- `assets/models/enemies/sentry_*.glb` source readme says personal and commercial use is allowed, with no attribution required.
- `assets/models/weapons/energy_sword_laser.glb` was converted from the local LaserSwords source pack.
- `assets/models/loot/sci_fi_chest.glb`, `health_pack.glb`, `mine_pickup.glb`, `grenade_pickup.glb`, `ammo_small.glb`, `keycard.glb`, `syringe.glb`, and `assets/models/environment/kit_*.glb` are from Quaternius Sci-Fi Essentials Kit, CC0 1.0.
- `assets/models/enemies/essentials_*.glb`, `assets/models/weapons/essentials_*.glb`, and extra ammo pickups are from Quaternius Sci-Fi Essentials Kit, CC0 1.0.
- `assets/models/enemies/megakit_*.glb` and `assets/models/environment/megakit_*.glb` are from Quaternius Modular SciFi MegaKit, CC0 1.0.
- `assets/models/environment/industrial_*.glb` are from Industrial Cyberpunk 3D Tilepack by Slippers, CC0 1.0.
- `assets/models/environment/corridor_kitjam_2021_08.glb` is from the local "corridor level kit - dueddel" source folder. No obvious license file was present in the added folder; keep this as a candidate asset until license/source notes are confirmed.
- `assets/audio/**` attribution is tracked in `assets/audio/ATTRIBUTION.md`.

Keep source packs out of this runtime tree. Commit only curated, browser-ready GLB, texture, and OGG assets.
