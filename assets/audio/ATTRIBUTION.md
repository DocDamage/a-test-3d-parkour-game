# Audio Attribution

Runtime audio files in this folder are compressed OGG Vorbis derivatives of the local source packs under `my game assets/audio`.

Known license notes from the included source packs:

- General audio source folder: Creative Commons Attribution 4.0 International.
- Universal UI Soundpack: created by Nathan Gibson, licensed under Creative Commons Attribution 4.0 International.

Keep this file updated when adding new third-party audio packs. Source WAV/MP3/OGG files stay out of the runtime tree; only curated browser-ready clips should be committed under `assets/audio/`.
